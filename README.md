## Getting starting

### Tools

Check that [`Nodejs`](https://nodejs.org/en/download/) is installed :

```sh
$ node -v
```

### Install all dependencies

Install all dependencies :

```sh
$ npm install
or
$ yarn install
```

### Test
Execute the tests :
```sh
$ yarn test
```
